# IndirectX

DirectX managed wrapper for .NET

Provides low-level access to Direct3D APIs.

# Installation

For general use:

```shell
dotnet add package IndirectX.D3D11
```

or specify other packages as needed:

```shell
dotnet add package IndirectX
dotnet add package IndirectX.Dxgi
dotnet add package IndirectX.D3DCompiler
```

and if you need more advanced functionality:
```shell
dotnet add package IndirectX.Helper
```
